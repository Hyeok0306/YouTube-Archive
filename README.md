

# YouTube-Archive

YouTube-Archive는 사용자가 관심 있는 유튜브 영상을 간편하게 검색하고, 
로컬 데이터베이스에 아카이빙하여 오프라인 상태에서도 학습 및 관리할 수 있도록 설계된 개인용 유튜브 아카이브 관리 앱입니다.
이 앱은 키워드 검색과 URL 직접 입력 방식을 지원하며, Room 데이터베이스를 통해 영상을 영구적으로 저장하고 관리합니다.

프로젝트는 MVVM 아키텍처와 Jetpack Compose를 기반으로 설계되었으며, 
데이터와 UI의 효율적인 결합을 구현하였습니다.
